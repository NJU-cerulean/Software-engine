- 这是通过java对软件的一种实现
- 有直接的命令行版本，也有一个UI的版本，其中UI是通过copilot生成的，用于展示
  
# 如何运行
  
- java ： 21+
- Maven ： 3.8+
  
运行方式：
  
1. `mvn -DskipTests clean compile`  
2. 运行命令版： `mvn --% exec:java -Dexec.mainClass=com.marketplace.Main`
3. 运行GUI版： `mvn --% exec:java -Dexec.mainClass=com.marketplace.ui.AppGUI`
